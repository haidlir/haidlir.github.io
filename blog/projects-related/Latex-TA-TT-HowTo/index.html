Template Latex Buku Tugas Akhir Teknik Telekomunikasi Universitas Telkom - HowTo

Sun, Jul 26, 2015

Berawal dari susahnya membuat buku tugas akhir menggunakan libre office, akhirnya aku berkenalan dengan Latex, sebuah program komputer untuk typesetting dokumen sekaligus markup language. Aku akhirnya memutuskan untuk mengulangi pembuatan buku tugas akhir menggunakan Latex setelah google mengantarkanku ke beberapa forum, yang sedikit banyak memmbuatku kepo tentang apa itu Latex. Banyak tulisan maupun komentar di forum-forum yang aku kunjungi itu menggambarkan bagaimana mudahnya memanfaatkan Latex sebagai word processor. Namun, aku beri descalimer dulu, bahwa kemudahan itu datang setelah berjuang beberapa saat memahami cara kerja Latex, dan membuat template yang cocok dengan yang diinginkan. Walaupun begitu, tidak perlu takut bergelut dengan Latex, ribuan bahkan mungkin jutaan laman forum, serta pengguna Latex lainnya siap membantu untuk melewati segala keriwehan saat menggunakan Latex. Sehingga pada akhirnya, ekspektasi menggunakan Latex untuk memudahkan dalam pembuatan tugas akhir maupun karya ilmiah lainnya dapat menjadi kenyataan.

Di Internet banyak sekali tulisan yang membahas tentang Latex. Namun, aku sarankan dua laman ini 1 dan ini 2 jika anda ingin mengetahui sekilas tentang Latex. Sedangkan untuk belajar kode-kode Latex yang sering digunakan seperti penebalan, pemiringan huruf, dan lain sebagainya dapat dilihat di cheatsheet berikut dan Latex Wikibooks.

Setelah menyelesaikan tugas akhirku, aku terpikirkan untuk membuat template Latex untuk buku tugas akhir. Jadi bagi para pejuang tugas akhir lain yang ingin memanfaatkan Latex dalam pembuatan buku tugas akhir, dipersilahkan untuk menggunakan tempate yang sudah aku buat dan unggah ke https://github.com/haidlir/Latex_TA_S1TT_TelU. Template tersebut merupakan dasar template yang aku gunakan juga dalam menyusun tugas akhirku. Oleh karena itu, template ini memang disesuaikan dengan format yang digunakan di prodi S1 Teknik Telekomunikasi Universitas Telkom. Namun bagi mahasiswa lain yang ingin menjadikan template ini sebagai dasar dari karya ilmiah lainnya dipersilahkan, tentunya tak melupakan aturan lisensi yang dari awal dibuat oleh Erik Dominikus, pembuat template awalnya.

Tulisan ini dibagi ke dalam beberapa bagian:

  • Install Latex ke dalam komputer,
  • Download/Clone Template Latex untuk Buku Tugas Akhir,
  • dan Keterangan Template Latex untuk Buku Tugas Akhir.

Install Latex

Untuk menggunakan Latex, pada komputer harus terpasang/ter-install Latex.

$ sudo apt-get install texlive

Terkadang saat melakukan proses install texlive, kita diberikan pesan error bahwa repositori tidak tersedia, oleh karena itu silahkan tambahkan link repositori untuk texlive.

$ sudo add-apt-repository ppa:texlive-backports/ppa
$ sudo apt-get update
$ sudo apt-get install texlive

Untuk memudahkan dalam proses editing file Latex dan viewing hasil kode Latex yang dibuat, aku sarankan anda menginstall text editor yang sudah dilengkapi plugin Latex. Aku sendiri menggunakan TexMaker [1] sebagai editor. Cara meng-install TexMaker sebagai berikut.

$ sudo apt-get install texmaker

Untuk pengguna sistem operasi selain Ubuntu silahkan merujuk ke web masing-masing aplikasi yang dibutuhkan, web texlive dan web texmaker

Download/Clone Template Latex

Sebelumnya, aku telah membuat sebuah mengunggah template yang aku buat ke github. Oleh karena itu, bagi anda yang ingin menggunakan template ini dapat melakukan cloning repositori template ini.

$ git clone https://github.com/haidlir/Latex_TA_S1TT_TelU.git

atau anda dapat mendownload file .zip, lalu anda dapat melakukan ekstraksi file .zip tadi. Template dapat diunduh di sini.

Dokumentasi Template Latex untuk Buku Tugas Akhir

Setelah template sudah berada di local atau komputer anda. Anda dapat memulai proses pembuatan buku tugas akhir anda.

Files yang harus di-edit untuk menyesuaikan template ini menjadi buku yang anda inginkan adalah konfigurasi.tex. Seusaikan nilai setiap variabel dengan tugas akhir anda.

\var{\judul}{[Judul Tugas Akhir dalam bahasa Indonesia]}
\Var{\Judul}{[Judul Tugas Akhir dalam bahasa Indonesia]}
\var{\judulInggris}{[Judul Tugas Akhir dalam bahasa Inggris]}
\Var{\JudulInggris}{[Judul Tugas Akhir dalam bahasa Inggris]}
\var{\type}{Tugas Akhir}
\Var{\Type}{Tugas Akhir}
\var{\penulis}{[Nama Penulis]}
\var{\alamat}{[Alamat rumah Penulis]}
\var{\tlp}{[Nomor kontak Penulis]}
\var{\email}{[Email Penulis]}
\Var{\Penulis}{[Nama Penulis]}
\var{\nim}{[Nim Penulis]}
\Var{\Fakultas}{[Fakultas Penulis berkuliah]}
\var{\fakultas}{[Fakultas Penulis berkuliah]}
\Var{\Program}{[Prodi Penulis berkuliah]}
\var{\program}{[Prodi Penulis berkuliah]}
\Var{\Tahun}{[Tahun Publikasi tugas akhir]]}
\var{\gelar}{[Gelar yang akan diraih Penulis]]}
\var{\tanggalPengesahan}{[Tanggal pengesahan tugas akhir]} 
\var{\pembimbingSatu}{[Nama Pembimbing 1]}
\var{\nikSatu}{[NIP Pembimbing 1]}
\var{\pembimbingDua}{[Nama Pembimbing 2]}
\var{\nikDua}{[NIP Pembimbing 2]}
...
\Var{\kataPengantar}{Kata Pengantar}
\Var{\babSatu}{Pendahuluan}
\Var{\babDua}{Tinjauan Pustaka} -> Ganti dengan nama bab yang diinginkan.
\Var{\babTiga}{Menggunakan \latex} -> Ganti dengan nama bab yang diinginkan.
\Var{\babEmpat}{Template \latex~ untuk Tugas Akhir S1 Teknik Telekomunikasi} -> Ganti dengan nama bab yang diinginkan.
\Var{\babLima}{Kesimpulan dan Saran} -> Ganti dengan nama bab yang diinginkan.

Setelah anda merubah nilai variabel di dalam konfigurasi.tex, lanjutkan dengan pengisian konten buku anda. Silahkan modifikasi file

  • abstrak.tex
  • abstract.tex
  • pengantar.tex
  • bab1.tex
  • bab2.tex
  • bab3.tex
  • bab4.tex
  • bab5.tex
  • pustaka.tex

Dengan menggunakan Latex, anda tidak perlu ribet membuat daftar isi, gambar, dan tabel. Setelah kode-kode Latex diproses dua kali tanpa pesan error, biasanya halaman daftar isis, gambar, dan tabel sudah secara otomatis terbuat. Begitu juga dengan halaman pengesahan dan orisinalitas, sudah secara otomatis terbuat sesuai dengan nilai variable yang dimasukkan ke konfigurasi.tex tadi.

Beberapa kekurangan dari template ini yang sudah terdeteksi dan masih akan diperbaiki antara lain.

  • Halaman lampiran yang belum terintegrasi dengan template yang dibuat, sehingga harus dimasukkan atau digabungkan secara manual.
  • Oleh karena itu, halaman daftar lampiran juga tidak otomatis terbuat dengan benar, sehingga harus diisi secara manual.
  • Pemenggalan kata masih diaktifkan, karena jika tidak diaktifkan, maka listing kode yang dimasukkan ke dalam dokumen Latex tidak bisa melakukan breakline. Namun anda dapat mematikan fitur pemenggalan kata dengan merubah bari ke 275 pada ta.sty dari
%\usepackage[none]{hyphenat}

menjadi

\usepackage[none]{hyphenat}

Bagi kawan-kawan lain yang ingin bertanya maupun memberi kritik dan saran, dipersilahkan menghubungi penulis.
